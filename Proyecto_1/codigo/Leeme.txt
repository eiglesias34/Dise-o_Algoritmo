El código del proyecto esta comprendido por dos archivos de código en c++:

	- main.cpp: Programa principal en donde se realiza la lectura del archivo de
				entrada, la creación de estructuras necesarias y se ejecuta la 
                llamada al algoritmo encargado de hallar el camino aproximado.

	- grafo.cpp: Script con las estructuras y funciones necesarias para 
                 representar y operar con nodos, aristas, y grafos

Ejecución
 
Para ejecutar la aplicación debe unicamente compilar el archivo main.cpp usando
el estandar c++11. Para ello ejecute desde el terminal en un sistema Unix:

	$g++ main.cpp -o main -std=c++11

con esto debería tener su programa compilado y tener en el directorio de 
compilación un arhcivo ejecutable llamado main.

Luego, para correr la aplicación ejecute el siguiente comando:

	$./main nombre_archivo_entrada

en caso de no indicar un archivo de entrada saldrá un mensaje de error por pantalla.
